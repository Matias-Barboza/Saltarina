# Saltarina
Es un videojuego de plataformas 2D, desarrollado en el curso de introducción al desarrollo de videojuegos Pixel Lab.

## Desarollado en:
- Motor de videojuegos:
  - Godot.
- Lenguaje de programación:
  - GdScript.
